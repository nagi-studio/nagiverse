# Graphology 笔记

Graphology 是一个强大的 JavaScript 图数据结构库。

## 核心 API

- `graph.addNode(key, attributes)`
- `graph.addEdge(source, target, attributes)`
- `graph.forEachNode(callback)`

适合配合 [[sigma-graph-intro]] 做前端可视化。

也可以看看 [[@alice]] 关于 [[svelte-reactivity]] 的笔记，了解如何在 Svelte 中集成。
